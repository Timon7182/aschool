# A School — бриф на разработку сайта

## Цель
Сайт услуг образовательного центра **A SCHOOL** (Казахстан). Задачи: привлечение новых клиентов, посадочные страницы под контекстную рекламу (Яндекс.Директ / Google Ads). Значит: быстрая загрузка, чёткие CTA «Бесплатный пробный день», конверсионная форма заявки, SEO.

## Технологии
- Чистый статический сайт: HTML + CSS + vanilla JS. Без фреймворков и сборщиков.
- Хостинг: GitHub Pages (репозиторий `aschool`, базовый URL `https://timon7182.github.io/aschool/`).
- ВАЖНО: все пути между страницами и к ассетам — ОТНОСИТЕЛЬНЫЕ (сайт живёт в подпапке /aschool/).
- Google Fonts: Nunito (600–900), Manrope (400–800), Material Symbols Rounded.

## Дизайн-система (строго следовать референсам!)
Референсы в `design-reference/`:
- `landing.dc.html` — главная страница (дизайн-макет)
- `program-prep.dc.html` — страница программы «Подготовка к школе» (дизайн-макет)

Это шаблоны Claude Design (`.dc.html`): inline-стили, `{{ … }}`-плейсхолдеры, `sc-for`/`sc-if` циклы, данные в `<script data-dc-script>` внизу. Их нужно ПЕРЕНЕСТИ в обычный семантический HTML/CSS: развернуть циклы в статичную разметку с данными из скрипта, `style-hover`/`style-focus` → CSS-классы с :hover/:focus, модалка и FAQ-аккордеон → vanilla JS.

Палитра: фон #FDFCFF, текст #241E38, вторичный #6B6580, фиолетовый #6C4BC7 / тёмный #4C2A85, зелёный (CTA) #1F8A5B, акценты как в макетах. Скругления 16–36px, мягкие тени rgba(76,42,133,…).

`x-import image-slot` (фото-слоты) → красивые CSS-заглушки: градиент в палитре + Material-иконка + подпись, с теми же размерами/скруглениями. Разметить `<img data-photo-slot>`-friendly структурой, чтобы владелец потом легко подставил реальные фото.

## Яндекс.Метрика
В `main.js`: `const YM_COUNTER_ID = null; // TODO: вставить номер счётчика` — сниппет Метрики инициализируется только если ID задан. На каждый клик по соцсетям/телефону — вызов цели (безопасно, если ym не инициализирован — проверять typeof).

## SEO (обязательно)
- Уникальные `<title>` (≤60 зн.) и `<meta description>` (≤160 зн.) на каждую страницу, с ключами: «подготовка к школе», «образовательный центр», «продлёнка», «мектепке дайындық», город.
- `<link rel="canonical">`, hreflang (ru/kk/x-default), Open Graph + Twitter Card.
- JSON-LD: на главной — `EducationalOrganization` (name, url, sameAs соцсети, телефон); на странице программы — `Course` + `FAQPage` (из FAQ-секции).
- Семантика: один `<h1>`, `<header>/<nav>/<main>/<section>/<footer>`, alt у изображений, aria-атрибуты у аккордеона/модалки/бургера.
- Мобильная версия обязательна: бургер-меню, сетки схлопываются, кликабельный телефон.
